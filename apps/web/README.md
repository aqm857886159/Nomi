# Nomi Web

Nomi Web is the local workspace for script writing, image generation, video generation, editing, and preview.

## Stack

- React 18
- TypeScript
- Vite
- Mantine
- React Flow
- Zustand

## What it does

- Script to node creation
- Image and video generation
- Timeline editing
- Asset reuse across canvas and timeline
- Local-first project storage

## Run

```bash
pnpm dev:web
```

## Test

```bash
pnpm test:web
```

## Design

Follow the Nomi design docs and tokens in this repo.
